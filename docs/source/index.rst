PyCharge Documentation
======================

PyCharge is a differentiable electromagnetics simulation library for moving point charges built on JAX.

------------

   :gutter: 2

Installation
------------

PyCharge is available on `PyPI <https://pypi.org/project/pycharge>`_ and can be installed with:

.. code-block:: bash

    pip install pycharge
